# Agenda app
Simple react native agenda app.

## To use it:

### On local server
#### Requirements:
1. expo-cli
2. nodejs
3. npm

#### Installation:
1. git clone https://github.com/MMAGANADEBIA/react-native-agenda.git
2. cd react-native-agenda
3. npm install 
4. expo start

### apk
#### Requirements
- Android phone

#### Installation package:
- Available on release page in this repository.
